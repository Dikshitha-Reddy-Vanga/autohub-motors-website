# AutoHub Motors

A responsive static website for a second-hand car showroom where users can explore available cars, view pricing details, and submit inquiries.

---

## Project Overview

AutoHub Motors is a simple and user-friendly front-end web application built using HTML and CSS. It showcases pre-owned cars along with their details and provides an easy way for users to contact the showroom.

---

## Features

* Responsive design (mobile-friendly layout)
* Navigation bar for easy section access
* About and Services sections
* Car gallery with hover effects
* Pricing table for available cars
* Contact form for customer inquiries
* Clean and modern UI design

---

## Tech Stack

* HTML5
* CSS3
